# SMC/ICT策略分析提示词优化测试部署

## 部署目标
基于原有btc_trading_bot.py，优化SMC/ICT策略分析提示词，解决以下问题：
1. 提示词过于复杂，包含大量变量和条件
2. 专业判断权限描述模糊，缺乏明确定义标准
3. 技术指标限制过严，影响交易信号质量
4. 数据处理与提示词混合，不利于维护

## 优化方向
1. 简化提示词结构，保留核心分析要素
2. 明确定义AI专业判断权限和标准
3. 调整技术指标限制，提高灵活性
4. 分离数据处理代码与提示词定义

## 文件说明
- `1.env`: 环境变量配置文件（从原部署复制）
- `requirements.txt`: Python依赖包列表（从原部署复制）
- `btc_trading_bot.py`: 主要交易机器人文件（待优化）
- `optimized_smc_prompt.py`: 优化版SMC/ICT策略分析提示词
- `btc_trading_bot_optimized.py`: 集成优化提示词的修改版交易机器人
- `test_optimized_prompt.py`: 测试脚本，验证优化后的提示词功能
- `INTEGRATION_GUIDE.md`: 优化版提示词集成指南

2. 设计新的提示词框架
3. 实现提示词优化
4. 测试验证优化效果
